For


For se usa generalmente cuando quieres contar e iterar soobre una secuencia de numeros o para recorres arrays con indices, es mas adecuado cuando sabes cuandotas veces quieres que se ejecute el bucle.
en resumen es mas adecuado cuando quieres controlar el numero de iteraciones

EJEMPLO 1

Estrucuras basica de un ciclo for imprime los nuemos del 1 al 10


                <?php
                // Imprime los números del 1 al 10
                for ($i = 1; $i <= 10; $i++) {
                    echo $i . "
";
                }
                ?>

EJEMPLO 2

Encuentra los numeros primos del 1 al 50


                <?php
                function esprimo($numero){ // creamos las funcion esprimo y que requiere del la variable numero
                    if($numero <= 1) return false; // verificamos que sea numeros naturales mayores a 1
                    // si el resultado es menor a 1, automaqticamente retorna falso
                    for ($i=2; $i <= sqrt($numero); $i++) { // itera desde 2 hasta la raiz cruadada del numero buscado
                        // esto es por que lo divisores siempre estaran entre 2 y la raiz cuadrada del numero
                        if ($numero % $i ==0) return false;  // si alguno de los numeros es divisible quiere decir que tiene mas divisores aparte de si mismo, por lo tanto automaticamente nos regresa false.
                    }
                    return true;// si no se sumple lo anteriro llega a este punto en el cual nos retorna verdadero
                }
                 //NOTA return sn php y la mayoria de lenjuages termina automaticamente la funcion
                
                for ($i=1; $i <=50 ; $i++) { // lo primero que hacemos es recorrer el for del numero 1 al 50
                    
                    if(esprimo($i)){ // verificamos SI la funcion esprimo arroja un resultado verdadero
                        echo $i . "
"; // si da un resultado verdadero mostramos el numero
                    }
                }
                ?>

While y do While

While y do While viene siendo el mismo bucle con la diferencia de que do while se ejecutara minimo una vez

EJEMPLO while 

                <?php
                $numero = 0;

                while ($numero > 0) {
                    echo "Número: $numero
";
                    $numero--;
                }
                ?>
EJEMPLO DO While 

                <?php
                $numero = 0;

                do {
                    echo "Número: $numero
";
                    $numero--;
                } while ($numero > 0);
                
                ?>
Cuando usar for y while

FOR: es util cuando sabes cuantas veces se debe iterar sobre un rango numerico
ejemplo: (contar numeros, recorrer arrays idexados, iteraciones fijas.)

WHILE Se usas cuando no sabes cuantas veces se debe ejecutar el bucle y depende de un condicion externa
Ejemplo: Esperaar un evento, repertir hasta que una condicion se cumpla

Bucles
Break y Continue

break: Sale del bucle y no se ejecutan mas iteraciones despues del break 


                    <?php
                    // rompe el ciclo cuando llega ala iteracion numero 5
                    for ($i = 1; $i  < 10; $i++) {
                        if ($i == 5) {
                            echo $i . "
";
                            break;
                        }
                        echo $i . "
";
                    }
                    ?>
continue: Salta a la siguiente iteración del bucle y las iteraciones restantes aun se ejecutan 


                    <?php
                    //continua si i es igual a 5 salta la iteracion y continua el ciclo
                    for ($i = 1; $i <= 10; $i++) {
                        if ($i == 5) {
                
                            continue;
                        }
                        echo $i . "
";
                    }
                    ?>

Alcance de las variables (Scope)

El alcanze de una variable de termina donde puede ser usada
las variables que se declaran dentro de la funcion son locales y solo pueden usarse dentro de las funcion
las variables que se declaran fuera de la funcion son Globales, se pueden mandar a llamar desde cualquier parte
si quitarmos la palabra goblal nuestra variable X ya no seria accesible


                    <?php
                    $x=10;// variable global

                    function mostrarX(){
                        global $x; //hace que la variable sea accesible en la funcion ;
                        echo $x;
                    }
                
                    mostrarX(); // resultado 10
                    ?>
